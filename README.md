# Administrador-De-Noticias
administrador de noticias institucionales


////Notas////
una usuario no puede validar su propia noticia, y un usuario no puede editar una noticia que no sea suya 
las noticias se comprueban si estan expiradas siempre que un usario ingresa a la pagina, o cambia un parametro
Solo los usuarios con el rol de ADMINISTRADOR pueden editar estos parametros, este rol solo puede ser concedido mediante la BD
